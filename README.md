
# Changen Yang Homepage

Personal academic homepage for Changen Yang.

## Structure

- `contents/`: markdown and YAML content
- `static/`: CSS, JavaScript, and site assets
- `index.html`: main page entry

## Update Content

- Edit profile and contact information in `contents/home.md`
- Edit publications in `contents/publications.md`
- Edit research projects in `contents/project.md`
- Edit awards in `contents/awards.md`
- Edit global text in `contents/config.yml`

## Preview And Export PDF

- Run `.\preview.ps1` from the project root.
- Open `http://localhost:8000` in a browser.
- Use the `Print / Save PDF` button or the browser print dialog.
- The site now includes a print stylesheet optimized for A4 PDF export.
